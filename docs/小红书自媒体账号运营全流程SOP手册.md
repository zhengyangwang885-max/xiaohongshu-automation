# 小红书自媒体账号运营全流程 SOP 手册

> 从账号定位 · 内容生产 · 日常运营 · 引流转化 · 平台规则 到 数据复盘的一站式标准作业流程。
> 本手册整合团队现有《全网短视频执行手册》《七天养号全流程》《账号定位表》《爆款标题十大模版》《账号自检清单》《矩阵获客 / 私域运营》等前期资料，并结合网盘《03-小红书课程 / 3-文字资料》中的平台规则（《小红书引流SOP》《小红书问答》《小红书常见问题》《运营高危词汇表》等）重新落地。

## 使用说明

**怎么用这份手册：**

- **新账号**：按【第二章 前期筹备】→【第六章 0–30 天起号时间表】逐项打钩执行。
- **发布 / 限流出问题**：先查【第九章 平台最新规则】的限流自查 SOP 与违禁词，再回到内容模块整改。
- **批量做号 / 矩阵**：用【第七章 矩阵与引流转化】，配合每个账号一份【附录 A 账号定位表】。
- **多行业通用**：日用品 / 装修 / 法律 / 二手车 等通用流程一致，差异只在“定位 + 选题 + 话术”。

**小红书 vs 抖音：方法论平移对照**

| 环节 | 抖音原打法 | 小红书落地要点 |
|---|---|---|
| 核心载体 | 短视频 + 直播 | 图文笔记为主、视频笔记为辅；封面权重极高 |
| 流量逻辑 | 推荐流（完播 + 互动） | 推荐流 + 搜索流（关键词 + 收藏权重高，长尾流量大） |
| 人群 | 全年龄、下沉广 | 女性为主、一二线城市、25–40 岁消费力强 |
| 养号 | 刷推荐 + 直播 1 小时 | 模仿真人刷本领域 + 收藏评论；养标签，不需直播 |
| 主页 | 五件套（含视频封面） | 五件套（头像 / 背景 / 昵称 / 简介 / 置顶笔记） |
| 转化 | 直播 + 私域带货 | 私信 / 评论引导 + 主页留钩子 + 私域承接（严合规） |

## 第一章　运营整体框架

一个能持续产出咨询 / 成交的小红书账号，本质是把下面这条链路跑通、并标准化：

> **定位 → 养号 → 内容生产 → 发布 → 数据复盘 → 引流承接 → 转化 → 复制矩阵**

| 阶段 | 目标 | 关键产出物 | 对应章节 |
|---|---|---|---|
| 前期筹备 | 把账号“立住”，方向不跑偏 | 账号定位表、关键词库、对标拆解表、五件套 | 第二章 |
| 内容生产 | 稳定产出有流量的笔记 | 选题库、标题、正文、封面、脚本/排版 | 第三章 |
| 发布运营 | 把每篇笔记发好、维护好 | 发布清单、评论区运营、薯条投放 | 第四章 |
| 数据复盘 | 看懂数据、找到问题 | 周/月数据表、自检表、流量诊断 | 第五章 |
| 起号执行 | 0–30 天把号从 0 跑到稳定 | 起号时间表、每日动作清单 | 第六章 |
| 引流转化 | 把公域流量变成私域成交 | 引流路径、承接话术、矩阵 SOP | 第七章 |
| 合规风控 | 不违规、不限流、不封号 | 违禁词、避坑清单 | 第八章 |
| 平台规则 | 吃透官方规则与功能门槛 | 规则提炼、功能门槛表、限流自查 | 第九章 |

## 第二章　前期筹备 SOP（整合现有前期工作）

### 一、账号定位（四问定位法 + 小红书补充）

沿用团队现有《账号定位表》的四个核心问题，每个新账号开做前必须填完，未填不开工。

| 序号 | 定位四问 | 填写要点 / 示例（以日用品为例） |
|---|---|---|
| 1 | 我是谁 | 人设标签：长沙做日用品批发 10 年的薇薇；一句话能记住 |
| 2 | 我的客户是谁 | 宝妈 / 想做副业的 30–45 岁女性 / 实体店主，按权重从左到右排 |
| 3 | 我的核心产品 | 只主推 1–2 类：洗护套餐 / 直播带货带教，别什么都卖 |
| 4 | 客户需求 + 我们的优势 | 需求：省钱、靠谱货源、能赚钱；优势：源头厂家、20 年、包教 |

**小红书专属补充（在四问基础上再定）：**

- **内容赛道**：只选 1 条主赛道，垂直度决定标签精准度。
- **人设三件套**：身份（你是谁）+ 价值（能给粉丝什么）+ 反差/记忆点（凭什么记住你）。
- **目标人群画像**：性别、年龄、城市、消费力、痛点、她会搜什么词——直接决定选题和关键词。
- **差异化卖点**：对比 2–4 个对标账号，找出“别人没做或没做透”的点，作为账号主轴。

### 二、关键词库（五种淘词法平移）

小红书是“搜索 + 推荐”双引擎，关键词决定能不能吃到搜索长尾流量。沿用现有五类淘词法，建一张关键词总表。

| 关键词类型 | 说明 | 示例（日用品 / 副业方向） |
|---|---|---|
| 基础词 | 行业/产品本身 | 洗衣液、日用品批发、洗护、囤货 |
| 地域词 | 城市 + 行业 | 长沙日用品、长沙批发市场、湖南货源 |
| 品牌词 | 自有品牌/IP 名 | 咪都、薇薇老师、XX 严选 |
| 人群词 | 目标人群自称 | 宝妈副业、新手小白、实体店主、全职妈妈 |
| 场景/痛点词 | 用户会搜的问题 | 怎么找货源、副业做什么、洗衣液测评、被割韭菜 |

> 用法：每类淘 10–30 个，标题 / 正文 / 话题里自然铺词；用小红书“搜索下拉词”持续补词。

### 三、对标拆解（全域营销扒皮流程平移）

对标账号需同时满足：同人群、同产品、同打法、同区域（至少满足 2 条）。每个新号开做前拆解 3–5 个对标账号。

| 拆解维度 | 记录什么 | 我们要抄/改什么 |
|---|---|---|
| 账号名 + 简介 | 怎么取名、怎么留钩子 | 命名公式、简介结构 |
| 内容形式 | 图文 / 视频 / 合集占比 | 决定我们主攻哪种 |
| 爆款选题 | 点赞收藏 Top10 的笔记主题 | 选题库直接借鉴 |
| 封面风格 | 排版、字号、配色、模板 | 做我们的封面模板 |
| 标题套路 | 用了哪种标题公式 | 套用十大标题模版 |
| 更新频率 | 一周几更、什么时间发 | 定我们的发布节奏 |
| 数据表现 | 粉丝、赞藏比、评论区问什么 | 判断需求 + 找选题 |
| 引流路径 | 怎么把人导到私域 | 合规借鉴承接方式 |

### 四、账号搭建：主页五件套（平移并适配小红书）

主页是“第一眼信任”。五件套必须 100% 完善后再发第一篇笔记。

| 五件套 | 标准要求 | 示例 |
|---|---|---|
| 头像 | 真人正脸 / 清晰品牌 LOGO，高清、明亮、亲和 | 本人商业形象照 |
| 背景图 | 一句话讲清你是谁 + 能给什么 + 钩子，文字大而清晰 | “长沙｜10 年日用品源头厂家｜教你做副业” |
| 昵称 | 地区 + 行业 + 人设，全中文易搜易记，含关键词 | 薇薇｜长沙日用品批发 |
| 简介(个人介绍) | 我是谁 / 我的经验 / 我能帮你 / 行动指引，4 行内 | 见下方模板 |
| 置顶笔记 | 1 篇引流/自我介绍笔记置顶（发 20 条后开通置顶） | “新手做副业避坑指南”合集 |

**简介模板（4 句话）：**

1. 我是谁：我是薇薇，在长沙做日用品批发 10 年。
2. 我的经验/实力：合作工厂 50+，服务实体店主过万家。
3. 我能帮你：教你 0 基础找货源、做小红书 + 直播带货。
4. 行动指引：主页置顶有新手攻略，记得先看 → 关注不迷路。

> ⚠️ 简介内不直接放微信/电话/“加我”等敏感引流词；可用谐音或把小红书号改成自己 ID（详见第九章引流规则）。

### 五、养号（小红书版“七天养号”）

养号目标 = 让系统给账号打上精准标签 + 提升账号权重。靠“搜 + 看 + 互动 + 收藏”，不依赖直播。（注：未违规过的新号其实可不强制养号，但模仿真人行为更稳。）

| 天数 | 核心动作 | 要点 |
|---|---|---|
| 第 1–3 天 | 模仿真人刷本领域笔记，每天 30 分钟以上 | 看完整篇、停留、有移动轨迹最好；不乱刷无关内容 |
| 第 2 天 | 包装账号：完善五件套 100% | 头像/背景/昵称/简介到位 |
| 第 3–4 天 | 开始发干货笔记，连发 4–7 篇 | 每天 1 篇，间隔 24 小时以上 |
| 有小爆后 | 再发引流笔记（晒数据型） | 引流笔记 3–5 天 1 篇，其余发干货 |
| 蓝V号 | 认证成功后可直接发 3 篇干货再发引流 | 省去普通号养号期 |

**养号红线（每天都要守）：**

- 一机一卡一号，养号期不频繁切换账号、不连公共 WiFi、不改资料反复横跳。
- 不在养号期发广告、不刷与定位无关的内容（会打乱标签）。
- 互动都围绕你的“目标人群 + 赛道”，让系统认准你是谁。

## 第三章　内容生产 SOP（八模块平移到小红书）

沿用现有《全网短视频执行手册》八模块流程，把“拍摄/剪辑”模块调整为“封面 + 图文排版”为核心。

### 模块一：选题策划

**选题五库（建立你的素材弹药库）：**

- **痛点库**：用户最焦虑/最想解决的问题（怎么选、怕被骗、怎么省钱）。
- **热点库**：平台热点、节日、行业事件。
- **对标库**：对标账号的爆款选题（直接改编）。
- **问答库**：评论区/私信里用户反复问的问题——最稳的选题来源。
- **故事库**：你的真实案例、客户故事、踩坑经历。

**爆款选题公式：**

| 公式 | 套路 | 小红书示例 |
|---|---|---|
| 痛点 + 解决方案 | 戳痛点再给答案 | “做副业总被割韭菜？记住这 3 点” |
| 盘点合集 | N 个清单，便于收藏 | “长沙宝妈在家能做的 5 个副业” |
| 避坑 / 反常识 | 打破认知，制造好奇 | “买二手车千万别信这 4 句话” |
| 教程/干货 | 手把手，收藏率高 | “0 基础做小红书的完整流程” |
| 对比测评 | 帮用户做决策 | “5 款洗衣液实测，第 3 款绝了” |
| 身份共鸣 | 锁定人群引共鸣 | “35 岁全职妈妈，我靠这个月入过万” |

### 模块二：标题撰写（爆款标题十大模版）

平移现有《爆款标题十大模版》，叠加小红书标题特性：≤20 字最佳、关键词前置、可带 1–2 个 emoji、制造好奇或利益点。

| 标题模版 | 公式 | 小红书示例 |
|---|---|---|
| 疑问揭秘式 | 一个让人想点的问题 | 做小红书一个月能涨多少粉？💰 |
| 数字清单式 | 数字 + 干货 | 新手做账号必看的 7 个细节✅ |
| 人群+结果式 | 锁定人群+给结果 | 宝妈在家做副业，3 个月回本😭 |
| 痛点+预防式 | 点痛点+给方案 | 别再这样发笔记了！难怪没流量 |
| 反差/争议式 | 打破常规立观点 | 劝你别轻易做二手车⚠️ |
| 保姆级教程式 | 强调完整可抄 | 0 基础起号保姆级教程（建议收藏） |
| 亲测/真实式 | 我亲身经历 | 我用这个方法，笔记篇篇过千赞 |
| 福利/资源式 | 给好处 | 整理了一份新手避坑清单（免费） |

### 模块三：正文文案创作

**小红书正文黄金结构：**

1. **开头钩子**（前 1–2 行）：抛痛点/利益点/悬念，让人点开“展开”。
2. **正文主体**：分点列（1️⃣2️⃣3️⃣ 或小标题），每点短句，多换行，手机端易读。
3. **个人观点/案例**：加真实感和信任。
4. **结尾引导**：引导“收藏/关注/主页看更多”——注意不能用“留言送XX/关注抽奖”等诱导式表述（违规，详见第九章）。
5. **话题标签**：3–6 个，含核心关键词 + 人群词 + 热门话题。

> 文案语气：口语化、像朋友分享，不官方、不硬广。每段不超过 3 行。

### 模块四：封面设计（小红书命脉，重点）

小红书 70% 的点击由封面决定。必须做统一的封面模板。封面尺寸：图文必须 3:4（竖版，约 1080×1440），别用横版。

**爆款封面三要素：**

- **大字标题**：3–8 个字的核心利益点/钩子，占封面视觉中心，远看也清楚。
- **强对比配色**：背景干净、文字醒目；同一账号配色/字体/版式统一（强化人设）。
- **画面真实/精致**：真人出镜或实拍场景；干货类用“清单感”排版。

| 封面类型 | 适用内容 | 排版要点 |
|---|---|---|
| 大字报型 | 干货/教程/观点 | 纯色背景 + 超大标题 + 1 行副标题 |
| 清单合集型 | 盘点/避坑 | 标题 + 序号列表预览，营造“信息量大” |
| 真人出镜型 | 人设/经验分享 | 本人正脸 + 一句话标题，建立信任 |
| 实拍种草型 | 产品/好物 | 高清实拍 + 小字卖点，质感优先 |

### 模块五：脚本设计 / 图文排版

- **图文笔记**：一般 6–9 张图，第 1 张封面、第 2–N 张分点讲解，最后 1 张引导关注/合集。
- **视频笔记**：黄金 3 秒开头（先抛结论/痛点），结构 = 钩子 → 干货 → 引导，时长 15–60 秒为宜；视频封面竖版 9:16 或 3:4。
- 封面文字与正文标题错位互补，不要完全重复。

### 模块六：素材整理（五库）

素材五库：选题库、文案库、封面模板库、图片/视频素材库、爆款案例库。图片素材可用 ColorHub、Picjumbo；模板用稿定设计。统一云盘归档，命名：`日期_账号_选题`。

### 模块七：制作（图文排版 / 剪辑）

- **图文**：用稿定/醒图/Canva 做封面，套自家模板；图片统一 3:4 竖版（1080×1440）。
- **视频**：剪映剪辑，1080P、竖版 9:16，加字幕，配热门 BGM；导出去水印（第三方水印会被判站外引流）。
- **仿写爆款**：找 30 天内本领域爆款（评论几十条、有人求带），替换为自己的项目并做改动提高原创度，注意敏感词。

### 模块八：平台发布

详见第四章发布运营 SOP 与第九章平台规则。

## 第四章　发布与日常运营 SOP

### 一、发布前检查清单（每篇必过）

| 项 | 检查内容 | ✓ |
|---|---|---|
| 封面 | 3:4 竖版、大字标题清晰、模板统一、无错别字 | □ |
| 标题 | ≤20 字、含关键词、有钩子 | □ |
| 正文 | 开头钩子 + 分点 + 结尾引导，排版换行清爽 | □ |
| 话题 | 3–6 个，含核心词 + 人群词 + 热门话题 | □ |
| 定位 | 带城市定位（本地获客必开） | □ |
| 合规 | 无违禁/高危词、无诱导、无第三方水印 | □ |
| 查词 | 发布前用零克查词等工具过一遍敏感词 | □ |
| 时间 | 选目标人群活跃时段发布 | □ |

### 二、发布节奏

- 起号期：每天 1 篇，间隔 24 小时以上，稳定更新 ≥ 7 天。
- 稳定期：保证质量前提下越多越好，至少每周 2–3 篇，保持垂直。
- 黄金时段：上午 11–13 点、晚上 19–22 点（按后台粉丝活跃时段微调）。
- 发布后可修改，但最多 1–2 次，每次都会重新审核，改多了易判违规。

### 三、评论区 / 私信运营

- 发布后 1 小时是黄金互动期：自己/同事先评论引导话题，主动回复每条评论。
- 评论区埋钩子：用“主页看 / 已整理好”引导，绝不直接留联系方式，不“诱导”关注点赞。
- 私信承接走平台规则（详见第七、九章），首条话术统一、快速响应。

### 四、薯条 / 付费投放（可选）

- 自然流跑出小爆款（赞藏比高）的笔记，再追投薯条放大，别投没数据的。

## 第七章　引流转化与矩阵 SOP（私域承接平移）

### 一、合规引流路径

小红书对站外引流管控严格，核心思路是“让用户主动来找你”，并控制数量、分散风险。

| 路径 | 做法 | 合规/风险要点 |
|---|---|---|
| 私信引流 | 用户私信→回复联系方式 | 效率最高风险最高；单号每天 ≤3 个，多了先警告再禁言 |
| 主页钩子 | 背景图/简介/置顶埋“福利/攻略”诱因 | 简介不直接放微信，用谐音/公众号/把小红书号改成 ID |
| 评论区引导 | “主页看 / 已整理好”，小号“求带”引导 | 不在公开评论留联系方式、不诱导关注点赞 |
| 群聊承接 | 笔记关联群聊，群内小号引导 | 站内沉淀更安全；发引流贴前先建群放 5 个小号设管理员 |

### 二、私信承接话术（统一模板）

1. **快速响应（5 分钟内）**：“在的～看到你对XX感兴趣😊”
2. **自我介绍 + 挖需求**：“请问你是想了解XX方面吗？现在主要想解决哪块？”
3. **筛选精准用户**：“之前有没有了解过这个项目？未成年是做不了的，你这边多大了？”
4. **给价值 + 信任 + 报价**：发项目/产品详细介绍、能提供的服务与价值，再甩价格。

### 三、矩阵搭建 SOP

- 一个主号验证模型跑通后，再复制 3–N 个号做矩阵，分摊引流风险、放大流量。
- 矩阵号差异化：同赛道不同人设/角度/城市；回私信时用不同微信 ID，避免平台关联检测。
- 设备/IP 隔离：一机一号一卡，独立运营，内容去重不一键多发。
- 统一中台：选题库、封面模板、话术库共享，单号执行标准化。

### 四、转化闭环

> 公域内容 → 主页/评论钩子 → 私信承接 → 私域沉淀 → 信任培育 → 成交 → 复购/转介绍

## 第八章　合规与风控 SOP（避坑清单）

### 一、违规操作红线

- 不搬运/不一键多平台同发（去重、原创化）。
- 不频繁切号、不连公共 WiFi、不一机多号来回切换（会限流）。
- 不在评论/简介直接留联系方式；不诱导关注点赞留言（如“留言送XX”“关注参与活动”）。
- 不发第三方平台水印图、不盗用他人素材（新号发旧号删的内容可能判盗图）。
- 未成年人不单独出镜；不发违法违规、低俗内容。

### 二、禁言 / 处罚机制（务必牢记）

| 违规情形 | 后果 |
|---|---|
| 私信发联系方式被系统识别 | 账号限流约 2 周–1 个月不等 |
| 私信/留言违规第 1 次 | 禁言 48 小时后可解 |
| 第 2 次 | 禁言 7 天可解 |
| 第 3 次 | 永久禁言 |
| 评论违规 | 影响通常不大，但无法解除，之后注意 |
| 笔记严重违规 | 笔记不可见 / 账号被屏蔽 |

### 三、被限流/违规后的处理

1. **系统主动通知违规**：能改的改（如涂掉水印）；看不懂就死缠客服问哪里违规，再按提示改。
2. **系统通知“笔记生病/违规”但建议不要改**：直接删除，避免二次违规。
3. **没通知但数据异常**：走第九章“限流自查 SOP”。
4. **严重时**：暂停引流动作，先连发几篇安全优质笔记养回权重（一周以上）。

## 第九章　小红书平台最新规则提炼（官方 + 实操）

> 本章提炼自网盘《03-小红书课程 / 3-文字资料》中的《小红书引流SOP》《小红书问答》《小红书常见问题》《运营高危词汇表》等资料，是团队当前可依据的平台规则要点。平台规则日新月异，请以官方社区规范为准：<https://www.xiaohongshu.com/crown/community/rules> ，并每季度复核更新本章。

### 一、账号规则（权重 / 登录 / 资料）

| 事项 | 规则要点 |
|---|---|
| 登录设备 | 一机一卡一号；只能单点登录；同设备多个账号来回切换会被限流、判营销号 |
| 要不要养号 | 未违规的新号/老号其实不必刻意养号；违规过的号需养一段时间（一周以上） |
| 账号标签 | 平台按“单篇笔记”推荐，不会给账号打死标签；换领域可把旧笔记设为仅自己可见 |
| 删笔记 | 个位数删除无影响；大幅批量删会掉权重 |
| 改资料 | 改昵称/简介只要不含敏感违禁词，不影响权重；但频繁改“标签”会掉权重+限流，有流量后再改尤甚 |
| 实名认证 | 发笔记不需要；直播/带货/专业号/品牌合作需要；一个身份证只能认证一个账号 |
| 旧号违规 | 违规过的号注销后手机号一般可释放重注；处罚一般封号不封设备 |

### 二、功能权限门槛（粉丝/条件解锁）

| 评论置顶 | 单篇满 50 条评论解锁；或开通视频号即有 |
| 视频号功能 | 满 500 粉丝 |
| 视频自定义封面 | 开通视频号后解锁 |
| 专业号 | 实名认证即可开通 |

### 三、内容与发布规范

| 项目 | 规则 |
|---|---|
| 图文封面尺寸 | 必须 3:4 竖版（约 1080×1440），不要用横版 |
| 视频尺寸 | 横版 16:9；竖版 9:16 / 3:4。视频封面：横版 3:4、竖版 4:3 |
| 发布频率 | 无硬性限制，保证质量前提下越多越好；训练营建议每周至少 2–3 篇 |

| 发布后修改 | 可改，但最多 1–2 次，每次重新审核，改多了易被判违规 |

### 四、限流 / 违规自查 SOP（小眼睛 100 怎么办）

先分清是“没收录 / 审核中”“单篇限流”还是“账号限流”，再对症处理。

**自查五步：**

1. **查过审**：用小号看你主页这篇在不在；或发到微信点开看是否“审核中”。还在审核就耐心等。
2. **查收录**：把标题全文丢进小红书搜索框，搜不到说明没被收录 → 问客服。
3. **问客服**：帮助与客服 → 笔记申诉 → 选该笔记 → “笔记流量不好/搜不到” → 文字问“我这篇是否违规”。
4. **找人工**：双十一/节假日客服慢属正常；否则狂找人工客服。
5. **以上都没问题**：那就是内容不够好。换封面标题再试，或继续发下一篇。

| 类型 | 表现 | 自查重点 |
|---|---|---|
| 账号限流 | 不论内容好坏，所有笔记浏览量都很低 | 一机多号、昵称/头像/签名违规、旧号重登 |
| 单篇限流 | 这一篇和其他差距大、搜索搜不到 | 广告、诱导分享、买数据、非原创、违禁敏感词 |
| 非限流(权重低) | 数据忽高忽低、整体偏低 | 内容质量不稳定、用户反馈差，靠提质解决 |

> ⚠️ **重要**：私信发联系方式被识别 → 限流 2 周–1 月；禁言机制 48h → 7 天 → 永久（见第八章）。
